## Properties

### VisibleFilter

|Key|Value|Type|
|:--|:----|:---|
|id|The ID of this element|String|
|searchField|The ID of the search field|String|
|headlines|Elements of the headlines|Array/Object|
|datas|Elements of the rows and columns|Array/Object|
